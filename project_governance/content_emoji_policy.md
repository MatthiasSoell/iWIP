# Content Emoji Policy – SciBlog iWIP

Version: 1.0.0  
Scope: Blog-Inhalte, OER-Materialien und Präsentationen  
Status: verbindliche Content-Governance

---

## 1. Grundprinzip

Emojis werden im SciBlog iWIP **nicht dekorativ**, sondern **semantisch** eingesetzt.

Sie dienen dazu:

- Inhalte schnell zu strukturieren
- Orientierung zu erleichtern
- wiederkehrende Konzepte visuell zu markieren

Emojis stehen in der Regel **am Anfang eines Abschnitts oder Begriffs**.

---

## 2. Emoji-Konventionen

Die folgenden Emojis haben im SciBlog iWIP eine feste Bedeutung.

| Emoji | Bedeutung |
|------|-----------|
| 🧩 | Methode |
| 💻 | Medium |
| 🎨 | Gestaltung |
| 🤝 | Wirtschaft / wirtschaftliche Zusammenarbeit |
| 🧭 | Überblick / Systematisierung |
| 💭 | Fragestellung |
| 🧠 | Lernen / lernförderlich |
| 🎯 | Lehrziele |
| 👥 | Unterricht |
| 👩‍🏫 | Lehren |
| 💬 | Diskussion |
| 🌱 | Erweiterung |
| 🪞 | Reflexion |
| 🌐 | Plattform |

---

## 3. OER-Kennzeichnung

Für Inhalte, die als **Open Educational Resources** veröffentlicht werden,
wird folgende Emoji-Kombination verwendet:

🎓🌍🔓

Bedeutung:

| Emoji | Bedeutung |
|------|-----------|
| 🎓 | Bildung / Lehre |
| 🌍 | Offen zugänglich |
| 🔓 | Freie Lizenz |

Diese Kombination markiert Materialien, die unter einer offenen Lizenz stehen.

---

## 4. Einsatzregeln

- Emojis werden **sparsam eingesetzt**.
- In der Regel **maximal ein Emoji pro Absatz oder Abschnitt**.
- Emojis dienen der **Orientierung**, nicht der Dekoration.
- Die semantische Bedeutung der Emojis bleibt **konsistent**.

---

## 5. Geltungsbereich

Diese Konvention gilt für:

- Blogartikel
- OER-Seiten
- Präsentationen (Reveal)
- didaktische Dokumentationen

Sie dient der konsistenten Gestaltung des SciBlog iWIP.
